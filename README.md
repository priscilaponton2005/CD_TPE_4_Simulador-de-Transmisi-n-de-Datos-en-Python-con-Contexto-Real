# CD_TPE_4_Simulador-de-Transmisi-n-de-Datos-en-Python-con-Contexto-Real
# Simulador de Comunicación de Datos - UNEMI

## Descripción
Este proyecto consiste en un **Simulador de Comunicación de Datos** desarrollado en **Python**, diseñado como apoyo didáctico para la asignatura **Interacción Humano-Computador** de la Universidad Estatal de Milagro (UNEMI).
El simulador representa el proceso de transmisión de información desde un mensaje digital hasta su representación como una señal analógica, mostrando conceptos fundamentales de la comunicación de datos mediante una interfaz gráfica intuitiva.

## Características

- Interfaz gráfica desarrollada con Tkinter.
- Conversión de caracteres a representación binaria.
- Visualización de señales digitales.
- Simulación de señales analógicas.
- Monitor de tráfico en tiempo real.
- Explicaciones de escenarios reales de comunicación.
- Botones para transmitir, pausar, continuar y reiniciar la simulación.
- Visualización dinámica utilizando Matplotlib.

## Tecnologías utilizadas
- Python 3
- Tkinter
- Matplotlib
- NumPy

## Funcionamiento
1. Escribe un mensaje en el cuadro de texto.
2. Presiona **Transmitir y Analizar**.
3. El sistema convierte cada carácter a binario.
4. Se muestra:
   - La señal digital.
   - La señal analógica.
   - El escenario de aplicación.
   - La teoría relacionada.
5. El usuario puede pausar, continuar o reiniciar la simulación.

## Objetivo
Demostrar de forma gráfica y práctica cómo los datos digitales pueden transmitirse mediante señales analógicas, reforzando conceptos como:
- Comunicación de datos
- Señales digitales
- Señales analógicas
- Modulación
- Conversión de datos
- Representación binaria
- Aplicaciones reales de transmisión de información
